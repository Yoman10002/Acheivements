    Rooms

    Download server package

    Build a portable ZIP you can copy to any Windows PC. Includes the chat server, dashboard, and a one-click Start Chat Server.bat launcher. For a full .exe installer, install Inno Setup and run build\build-installer.ps1.

    Setup on any computer

    1. Install Node.js

    Download LTS from nodejs.org, or use the portable package from the Download tab.

    2. Start server

    npm install
npm start

    Chat: http://localhost:3000 — Dashboard: /dashboard

    3. Cloudflare Tunnel (internet access)

    cloudflared tunnel --url http://localhost:3000

    4. Migrate to a new server

    On the old server: Storage section → Export data folder (ZIP).

    On the new server: install the app, set storage path (or extract ZIP into data folder), then npm start.

    5. Run in background (optional)

    npm install -g pm2
pm2 start start.js --name chat
pm2 save
